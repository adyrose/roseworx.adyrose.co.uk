Icons

You can use any color modifier and size modifier to achieve the style you want. .--{size} .--{color}.

Size modifier classes include --xs, --sm, --lg, --xl

Color modifier classes include --black ,--grey ,--dark-grey ,--white ,--orange ,--red ,--blue ,--light-blue ,--turquoise ,--green ,--light-green ,--yellow ,--purple ,--pink ,--brown

Alternatively, manually give the i element a specific width and height to suit your needs.
