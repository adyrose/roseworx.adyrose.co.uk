buttons

The default button style is determined by the customisable button SCSS variables. however it is possible to change to color of any button by simply giving it a color modifier class .--{color}

Default

Inverted

Disabled

Any element

You can style any element like a button just by giving it a rwx-button class. You can aslo apply any color or variant modifier classes to it.

a.rwx-button [role="button"]
a.rwx-button.--orange [role="button"]
a.rwx-button.--inverted [role="button"]
div.rwx-button.--blue [role="button"]

p.rwx-button.--red.--inverted [role="button"]

No style

Sometimes its useful to use a button element for functionality and accessibility purposes but you dont wish it to be styled like a button. Add the no-decoration class to any button element
