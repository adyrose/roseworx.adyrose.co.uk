Grid Galleries

Maximum number of gallery items is 14. If you need more then start a new rwx-grid-gallery

Roseworx components have a scanDOM method which basically rescans the DOM for roseworx components and instantiates them. This is extremely useful for when you are dynamically / conditionally adding components to a page after the page load event or using Javascript frameworks like React or Vue. Simply run rwxGridGalleries.scanDOM().

By default this component will commence when it is scrolled in to view. You can override this by specifying the data-rwx-manual-control on the rwx-grid-gallery element. You can then manually control it by specifiying rwxGridGalleries.commence('ID') where 'ID' is the ID of the element to commence.

Description

The Roseworx Grid Gallery Component gives a three-dimensional entry animation which results in a unique, stylish and responsive picture grid gallery experience. Display images (with optional click throughs) and even add a caption overlay.

Usage
  • import rwxGridGalleries from 'roseworx/js/components/rwxGridGalleries';
  • Copy the html code and personalise it to your use case.
