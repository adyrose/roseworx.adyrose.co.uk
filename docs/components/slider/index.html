Slider

You can add a .--center modifier class to the .rwx-slider-slide node to make all the content inside vertically and horizontally aligned.

You can add a .--counter-circle class to the .rwx-slider node to make counters a circle (if data-rwx-slider-counters is specified).

Usage
import rwxSliders from 'roseworx/js/components/rwxSliders';

To programatically change slide simply call rwxSliders.goToSlide(id, slideNumber) - where id is the ID of the slider (an id must be present on the rwx-slider element) and slideNumber is the number of the slide you want it to go to.

Configuration Options

Place any of the following [data] attributes on the [rwx-slider] node to configure each slider the way you want.

data-rwx-slider-auto-slideThis will make the slider automatically slide on a timeout without any user interaction. You can specify this alone and the default timeout of 5 seconds will be used or you can specify your own timeout using data-rwx-slider-auto-slide="7" - where "7" is the time in seconds for example.
data-rwx-slider-countersThis will force the appearance of "counters" to appear in the slider which visually show how many slides there are and allows you to click through each slide.
data-rwx-slider-verticalSimply flips the slide direction from left->right to top->bottom
data-rwx-slider-reelingReeling means that if you were to go to a slide which isnt naturally the next one in order, all the other slides inbetween the current slide and the one you have gone to will be "reeled" through until the final one is reached.
SCSS Variables

Changing these variables does nothing unless the js is imported.

VariableDescription
$slider-counter-active-colorThe color of the counter when the corresponding slide is in view
$slider-counter-inactive-colorThe color of the counter when the corresponding slide is not in view
$slider-counter-sizeThe size of the counter
