Count-O-Meters

Count-O-Meters

The data-rwx-countometer-value="VALUE" attribute is required or nothing will be rendered.

Roseworx components have a scanDOM method which basically rescans the DOM for roseworx components and instantiates them. This is extremely useful for when you are dynamically / conditionally adding components to a page after the page load event or using Javascript frameworks like React or Vue. Simply run rwxCountOMeters.scanDOM().

By default this component will commence when it is scrolled in to view. You can override this by specifying the data-rwx-manual-control on the rwx-count-o-meter element. You can then manually control it by specifiying rwxCountOMeters.commence('ID') where 'ID' is the ID of the element to commence.

Description

The Roseworx Count-O-Meter Component visually animates, by way of color changing particles, a percentage from 0 up to a specified value. This component is responsive.

Usage
  • import rwxCountOMeters from 'roseworx/js/components/rwxCountOMeters';
  • Copy the html code and personalise it to your use case.
