Slideshow

You can use the arrow keys to navigate the slideshow.

The slideshow will fill the entire space of its parent element.

Description

The Roseworx Slideshow Component allows you to create a web based slideshow presentation easily with an 'audience grasping' gradient background, sleak transitions between slides and a three-dimensional aspect on the content which follows the mouse movements on the screen. This component is web accessible with keyboard navigation.

Usage
  • import rwxSlideshows from 'roseworx/js/components/rwxSlideshows';
  • Copy the html code and personalise it to your use case.
