Photo Tiles

Photo Tiles

You can even use an empty element with a solid background as a tile instead of an image.

Description

The Roseworx Photo Tile Component completely raises the bar when displaying a photo gallery or slideshow. It features an array of silky smooth transitions between pictures or solid color backgrounds and gives you the choice to change pictures manually or programatically. This component is web accessible with keyboard navigation.

Usage
  • import rwxPhotoTiles from 'roseworx/js/components/rwxPhotoTiles';
  • Copy the html code and personalise it to your use case.
Additional Methods

In order to use the additional methods, a unique id attribute must be present on the [rwx-phototile] element you are targeting. These must only be called on or after the window load event.

goToTile

void

(

ID Of Photo Tile Element: String
Number Of Tile: Number
Effect: String

)

Programtically change the current tile to the specified tile number.

						rwxPhotoTiles.goToTile('idOfTiles', tileNumber, effect);
Configuration Options

Optionally add any of these data attributes to the rwx-phototile element to configure it a certain way.

data-rwx-phototile-auto This will make the tiles automatically change on a timeout without any user interaction. You can specify this alone and the default timeout of 5 seconds will be used or you can specify your own timeout using data-rwx-slider-auto-slide='7' - where '7' is the time in seconds, for example.
data-rwx-phototile-no-thumbnails Will remove the thumbnail previews of each tile.
data-rwx-phototile-effect='effect' Specify here the animation effect that will happen between tiles, where 'effect' is one of the possible effects.
Possible Effects
  • bubble
  • spin
  • pixelated
  • slideRandom
  • slideLeft
  • slideRight
  • slideUp
  • slideDown
