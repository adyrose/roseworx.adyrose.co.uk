grid

You can change the default grid by adjusting the scss grid variables.

After 'md' breakpoint everything is full width and stacked by default.

You can put a rwxc-background-{color} on a .rwx-split-{measurement} and it will fill the whole size of the split. Alternatively put a div, with the .rwxc-background-{color} class on, inside the .rwx-split-{measurement} for it to fit the whole size minus the grid gutter.
