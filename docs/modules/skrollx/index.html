skrollx

Description

The Roseworx SkrollX Module gives you an incredibly easy way to transition elements in when they are scrolled in to view. SkrollX comes with lots of configuration options to personalize the scrolling experience.

Usage
  • import rwxSkrollX from 'roseworx/js/modules/rwxSkrollX';
  • Add a rwxsx attribute to any html element.
  • Add any of the following classes to the rwxsx element;
  • .rwxsx-start-left
  • .rwxsx-start-right
  • .rwxsx-start-bottom
  • .rwxsx-start-top
  • .rwxsx-start-initial
  • .rwxsx-start-top-left
  • .rwxsx-start-top-right
  • .rwxsx-start-bottom-left
  • .rwxsx-start-bottom-right
Configuration Options

Optionally add any of these data attributes to the rwxsx element to configure it a certain way.

data-rwxsx-delay='1'This will delay the animation of an element after it hits the trigger point- Where 'delay' is the delay time in seconds.
data-rwxsx-trigger='300'This will set the point at which elements will animate in- Where 'trigger' is the amount of pixels from the bottom of the screen.
