    skrollx

    SkrollX is a scrolling parallax library which allows you to animate dom elements in when they are scrolled into view.

    import rwxSkrollX from 'roseworx/js/modules/rwxSkrollX';

    Getting an element to animate in is extremely easy, simply give any element a rwxsx attribute and a class with any of the following options which specify its starting position;

    • .rwxsx-start-left
    • .rwxsx-start-right
    • .rwxsx-start-bottom
    • .rwxsx-start-top
    • .rwxsx-start-initial
    • .rwxsx-start-top-left
    • .rwxsx-start-top-right
    • .rwxsx-start-bottom-left
    • .rwxsx-start-bottom-right
    Configuration Options

    Place any of the following [data] attributes on the [rwxsx] node to configure each element the way you want.

    data-rwxsx-delay='1000'This will delay the animation of an element after it hits the trigger point- Where 'delay' is the delay time in seconds.
    Additional Methods
    setConfig

    void

    (

    Trigger: Number

    )

    Sets the global trigger point for all [rwxsx] elements. The 'trigger' is how far from the bottom of the page the top of an element has to hit before it is animated in to view. For example a trigger of window.innerHeight/2 will animate elements in when the top of each element hits to middle of the page vertically.
