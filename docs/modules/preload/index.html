    preload

    import rwxPreload from 'roseworx/js/modules/rwxPreload';

    You will need to add rwx-preload class to the body element of the page for this to work.

    The Roseworx Preload module solves the issue of CSS transitions running when a page is loaded.

    For example if an element has a css property of transform:translateX(-40px); & transition: all 1s ease;. When the page first loads, you will unintentionally see the element transition from its start position to its end position.
